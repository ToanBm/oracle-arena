## Games

### ⚔️ Prompt Gladiator (`/`)
A prompt engineering battle where players compete to write the most effective AI prompt. GenLayer validators act as decentralized judges.

**How it works:**
1. **Solo Mode**: Practice against static challenges (e.g., "Convince a guard to let you in").
2. **Multiplayer Battle**: Join a room and compete on a weekly AI-generated scenario.
3. **AI Judging**: Each validator runs its own LLM to score your response (1-10). If the majority of validators agree on the score (within ±2 points), the score is committed on-chain.
4. **XP Rewards**: Earn XP and climb the global leaderboard.

### 🌐 Oracle Arena (`/oracle`)
A multiplayer trivia game where the questions are generated by AI consensus on the blockchain — no human picks them.

**How it works:**
1. Host creates a room and shares a 6-digit code.
2. GenLayer validators generate 5 trivia questions via LLM and reach consensus on one set.
3. All players answer the same 5 questions within a 10-minute window.
4. XP is distributed by rank: 1st=100, 2nd=75, 3rd=50, 4th=25, rest=10.

## Quick Start

### Requirements
- [GenLayer Studio](https://studio.genlayer.com/) account (studionet)
- [GenLayer CLI](https://github.com/genlayerlabs/genlayer-cli): `npm install -g genlayer`
- Node.js 18+ and Bun/npm

### Setup & Deployment

```bash
# 1. Install dependencies
npm install
cd frontend && npm install && cd ..

# 2. Select network
npx genlayer network   # choose studionet

# 3. Deploy Both Contracts
# This uses the scripts defined in package.json
npm run deploy           # Deploys Prompt Gladiator (contracts/prompt_arena.py)
npm run deploy:oracle    # Deploys Oracle Arena (contracts/oracle_arena.py)

# 4. Initialize Prompt Gladiator (Mandatory for Solo Mode)
# Replace <PROMPT_CONTRACT_ADDRESS> with the address from step 3
npx genlayer write <PROMPT_CONTRACT_ADDRESS> seed_solo_challenges

# 5. Configure environment
cp frontend/.env.example frontend/.env
# Edit frontend/.env:
# NEXT_PUBLIC_CONTRACT_ADDRESS=<Prompt Gladiator Address>
# NEXT_PUBLIC_ORACLE_CONTRACT_ADDRESS=<Oracle Arena Address>

# 6. Start frontend
cd frontend && npm run dev
```

## GenLayer Development Notes (Fixing `DISAGREE` Errors)

When building Intelligent Contracts, ensure **absolute determinism**:

| Rule | Solution |
|---|---|
| **Deterministic Time** | Use `gl.message_raw["datetime"]` and parse to Unix seconds. Never use `gl.message.timestamp`. |
| **Minimal __init__** | Do not initialize `TreeMap` or write data in `__init__`. Use a separate `seed` function. |
| **TreeMap Usage** | Declare at class level. Reads default to empty. Avoid re-initialization. |
| **Consensus Speed** | AI-driven consensus takes ~2–5 min. Design UI with loading states. |

For detailed development guidelines, see [GEMINI.md](./GEMINI.md).
